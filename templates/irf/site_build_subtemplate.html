WELCOME PAGE

This page features Bootstrap's jumbotron, glyphicons and navbar.

JOB LIST

This page uses a ui-bootstrap datagrid to display data stored in a postgresql database. The grid is contextualized with CSS media queries. Data can be filtered by skill using Angular.js, or by company name with the help of an Angular typeahead. The visual filter is aided by CSS transition animations.

JOB DETAIL

This page is laid out with Bootstrap panels, filtered with Django conditionals. Data is pulled from postgresql using a Django queryset.

ABOUT

This page uses a ui-bootstrap carousel, sized with CSS media queries to fit your device.

PORTFOLIO

This page showcases CSS media queries and CSS animations, as well as Angular.js data binding.

All development for this site was done in a VMWare virtual machine running Ubuntu Linux. I used PyCharm as my IDE, with integrated GIT for version control. The site is deployed on Heroku, a PaaS that offers Django hosting.

Like most web projects, this one relied heavily on third party libraries, including:

  • Angular.js for interactive components
  • Boostrap for CSS
  • ui-bootstrap to integrate Angular and Boostrap without jQuery
  • postgresql as a database
  • Django to implement MVC and abstract the database
  • psycopg2 library to integrate Django and postgresql
  • virtualenv and virtualenv-wrapper for virtual environments
  • whitenoise for static file serving on Heroku

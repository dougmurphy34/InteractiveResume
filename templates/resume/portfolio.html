I believe the best websites are responsive websites. I use ootstrap to make this site look as good on a phone as it does on a computer. Try making your browser very narrow and watch the top nagivation menu. It changes to accomodate your situation.

ootstrap uses CSS media queries track your device information as it changes. Try it out:

  • You are using a small browser window (< 640 pixels wide).
  • You are using a medium browser window (640 to 1440 pixels wide).
  • You are using a large browser window (>1440 pixels wide).
  • This window is wider than it is tall.
  • This window is taller than it is wide.
